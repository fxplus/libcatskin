WebVoyáge Personal Search Preferences Help

Personal Preference let you define what your preferred search settings. Once configured, these settings will be in effect whenever you log in.

There are three sections of the Personal Search Preferences page which can be enabled in any combination. If one (or more) of these sections is not available for selection, it has not been enabled by your library. These 3 three sections are: Search options, Display options, and Databases .

Search options

  • Select a type of search and specify defaults for the selected search.

    • Display options

  • Select the desired number of records to display, per page, from the drop-down box.

    • Databases

  • Select all databases you wish to connect to when performing a search.
  • Remember that all of these databases (if you select more than one) will be connected to each time you log in and perform a search, until you go to the Databases page and change them, or change them here.

    • When finished, click the Save  button.
