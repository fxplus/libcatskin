Library Catalogue My Searches 

My Searches allows you to save up to 25 search queries to run again at a later time. You can save a search from the title list or from the search history page.

To execute a search from your list:

  1. Click Re-Run.

To get updates about new hits for a search:

  1. Set the update frequency and set Results to Y.
  2. Save your changes.
