WebVoyáge Basic Search Help

Basic search finds records using keywords located anywhere in a record, or only in specific fields.

To perform a basic search:

  1. Type in the search term(s) you want to find in the Search field. (Punctuation, case, and word order are ignored.)
  2. Select a different index from the next drop-down list to narrow or widen the scope of your search.
  3. Keyword searches allow using quotes to search a phrase, + to mark an essential term, * to mark an important term, or ? to truncate a word.
  4. Heading searches use a specific library categories.
  5. Command searches allow adding Boolean operators to keyword searches.
  6. Select from the Limit To: list to optionally limit your search by the available criteria.
  7. Click the Search button to begin your search.
